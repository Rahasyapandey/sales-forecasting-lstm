# Sales Forecasting AI 📈

A deep learning approach to time-series forecasting using LSTM (Long Short-Term Memory) networks. This project focuses on predicting future sales trends and visualizing model performance using **Matplotlib**.

## ✨ Key Features
* **LSTM Architecture:** Optimized for sequential data patterns.
* **Data Visualization:** Custom Matplotlib plots for Actual vs. Predicted values.
* **Scalable Logic:** Minimalist, comment-free core functions for better readability.

## 🛠️ Tech Stack
* **Python** (Core Logic)
* **TensorFlow/Keras** (Model Training)
* **Matplotlib** (Visualization)
* **Pandas/NumPy** (Data Processing)
